# Features & Roadmap

## Huidige Features

### Eigenaren
- [x] CRUD eigenaren
- [x] Postcode lookup (PDOK API)
- [x] UBN nummer voor fokkers
- [x] Zoeken op alle velden + patiënten

### Patiënten
- [x] CRUD patiënten via eigenaar
- [x] Diersoort, ras, geslacht
- [x] Geboortedatum met leeftijdsberekening
- [x] Chipnummer
- [x] Vachtkleur en vachttype
- [x] Gecastreerd/gesteriliseerd
- [x] Gewicht
- [x] Bijzonderheden

### Systeem
- [x] Livewire 3 componenten
- [x] Tailwind CSS (Vite build)
- [x] MySQL database
- [x] Nginx + SSL
